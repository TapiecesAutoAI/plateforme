# AI ENGINE

## Rôle

Le moteur IA est le cerveau de la plateforme.

Il analyse les demandes des utilisateurs et produit des recommandations justifiées.

## Responsabilités

- Comprendre les demandes des utilisateurs
- Identifier le véhicule
- Poser les bonnes questions
- Évaluer le niveau de confiance
- Recommander des pièces compatibles
- Expliquer chaque recommandation

## Principe

Le moteur IA est indépendant du frontend et du backend.

Il peut être utilisé par :

- le site web ;
- une application mobile ;
- une borne interactive ;
- une API partenaire.

## Règle fondamentale

L'IA ne recommande jamais une pièce lorsqu'elle ne dispose pas d'informations suffisantes.

Elle demande des précisions ou indique clairement son niveau d'incertitude.

## Objectif

Fournir des recommandations fiables, compréhensibles et vérifiables.
