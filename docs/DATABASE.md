# DATABASE

## Question

Quelles informations sont nécessaires au fonctionnement de la plateforme ?

---

# Objectif

Stocker uniquement les informations utiles afin de garantir des recommandations fiables et rapides.

---

# Véhicules

Chaque véhicule contient notamment :

* Marque
* Modèle
* Génération
* Année
* Motorisation
* Cylindrée
* Carburant
* Puissance
* VIN
* Plaque (si autorisée)
* Boîte de vitesses

---

# Pièces

Chaque pièce possède :

* Référence
* Fabricant
* Catégorie
* Description
* Compatibilités
* Prix
* Stock
* Garantie

---

# Compatibilités

Pour chaque pièce :

* Véhicules compatibles
* Restrictions éventuelles
* Conditions particulières
* Niveau de confiance

---

# Magasins

Chaque magasin possède :

* Nom
* Adresse
* Coordonnées
* Horaires
* Stock
* Délais

---

# Utilisateurs

Informations minimales :

* Nom
* Téléphone
* Email (si nécessaire)
* Historique des réservations

---

# Réservations

Chaque réservation contient :

* Client
* Pièce
* Véhicule
* Magasin
* Date
* Statut

---

# Historique IA

Pour améliorer le moteur IA :

* Questions posées
* Réponses du client
* Recommandation proposée
* Niveau de confiance
* Validation finale

---

# Principe

La base de données doit stocker des faits.

Les décisions sont prises par le moteur IA, jamais par la base de données.
