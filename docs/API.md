# API

## Question

Comment les différents composants de la plateforme communiquent-ils entre eux ?

---

# Objectif

Créer une architecture modulaire où chaque composant possède une responsabilité précise.

Les composants communiquent uniquement via des API clairement définies.

---

# Architecture générale

La plateforme est composée de plusieurs services :

- Interface Web
- Moteur IA
- Service Véhicules
- Service Pièces
- Service Compatibilité
- Service Stocks
- Service Réservations
- Base
