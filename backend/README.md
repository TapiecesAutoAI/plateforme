# BACKEND

## Rôle

Le backend est le cœur de la plateforme.

Il reçoit les demandes du frontend, interroge le moteur IA, consulte la base de données et renvoie les résultats.

## Responsabilités

- Gestion des utilisateurs
- Gestion des véhicules
- Gestion des pièces
- Gestion des réservations
- Communication avec le moteur IA
- Communication avec les services externes
- Sécurité
- Authentification

## Principe

Le backend contient toute la logique métier.

Aucune décision importante ne doit être prise dans le frontend.

## Évolution

Le backend doit pouvoir évoluer sans modifier le frontend ni le moteur IA.
